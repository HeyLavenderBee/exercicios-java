## Stream API
Esse diretório é um aprofundamento maior sobre interfaces funcionais, que já foram levemente retratadas no diretório `/src/interfaces`.
<br>
Dentro deste diretório, também existe uma subpasta `/desafio`, onde tem um arquivo .java resolvendo desafios propostos para praticar functional interfaces e vários métodos importantes.
<br><br>
Foi visto e estudado sobre as seguintes interfaces funcionais:
- `Consumer<T>()`
- `Supplier<T>()`
- `Function<T>()`
- `Predicate<T>()`
- `BinaryOperator<T>()`
- `Optional<T>()`


Existe um arquivo para o estudo cada uma delas, que estão no formato de exemplo: `ConsumerExxample.java`.
